Classical Problems

  • Coloring Problem
    • Find the lowest number of colors to color a map, with neighboring countries not being painted with the same color.
  • Knapsack Problem
    • Maximize the value of a knapsack in such a way that the quantity of items loaded in it does not exceed its weight limit.
  • Queen's Problem
    • Position the most number of queens on a chess board in such a way that they cannot attack each other.
  • Traveling Salesman Problem
    • Find the shortest route in which the salesman visits all cities only once.
